# Pokedex

<img src=".github/assets/pokedex.svg" />

A React + TypeScript application built with Vite for managing and viewing Pok√©mon data.

## Features

- Fast development with Vite and Hot Module Replacement (HMR)
- Type safety with TypeScript
- ESLint integration for code quality
- Support for React plugins using Babel and SWC

## Pre-installation

1. Install NodeJS through [NVM](https://github.com/nvm-sh/nvm)

2. Install yarn
   ```bash
   npm i -g yarn
   ```

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BuenClima/pokedex.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pokedex
   ```
3. Install dependencies:
   ```bash
   yarn install
   ```

## Usage

Start the development server:

```bash
yarn dev
```

Test the app:

```bash
yarn vitest
```

Open http://localhost:5173 in your browser to view the app.
